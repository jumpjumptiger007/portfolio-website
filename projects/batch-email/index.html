Key Components

1. Batch Email Sender (batch-email-smtp.py)

Sends marketing emails via Google Workspace SMTP with customizable options:

# Initialize the sender sender = BatchEmailSender( smtp_server="smtp.gmail.com", port=587, email="your.email@yourdomain.com", password="your-app-password" ) # Send batch emails results = sender.send_batch_from_csv( csv_path="recipients.csv", html_template=html_template, use_bcc=True, batch_size=50, check_unsubscribed=True )

2. Unsubscribe Handler (unsubscribe-handler.py)

Manages unsubscribe requests and maintains subscriber database:

@app.route('/api/unsubscribe', methods=['POST']) def unsubscribe(): email = data.get('email') reasons = data.get('reasons', []) # Update subscriber status cursor.execute( 'UPDATE subscribers SET subscribed = 0 WHERE email = ?', (email,) ) # Store unsubscribe reasons cursor.execute( 'INSERT INTO unsubscribe_reasons (email, reason, comments) VALUES (?, ?, ?)', (email, ', '.join(reasons), comments) )

Technical Highlights

  • Uses Python's smtplib for email sending
  • SQLite database for tracking subscriptions
  • Flask web service for handling unsubscribe requests
  • CSV-based recipient and mailing list management
  • Customizable email templates with personalization
  • Rate limiting to prevent email sending restrictions
